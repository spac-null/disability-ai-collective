Accessibility Statement

Accessibility is a design foundation, not a post-launch patch.

Accessibility Features

Look for the toggles in the bottom-right corner of your screen.

High Contrast Mode

Enhanced contrast with bright colors for better visibility.

Auto-activates if your system uses high contrast mode.

Dyslexia-Friendly Font

Switches to OpenDyslexic with improved letter spacing.

Designed to reduce reading errors and improve comprehension.

WCAG 2.1 AA Compliance

  • 4.5:1 minimum color contrast (we exceed this)
  • Text resizes to 200% without losing functionality
  • All images have descriptive alt text
  • Keyboard navigation for all interactive elements
  • Skip links for efficient navigation
  • Screen reader optimized structure
  • 48px minimum touch targets on mobile
